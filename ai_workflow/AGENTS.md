# AGENTS.md

## Agent Roles

## Proof Generator

Suggests Lean theorem statements and proof attempts from informal mathematical arguments.

## Verifier

Runs Lean and reports exact compiler feedback.

## Repairer

Modifies theorem statements or proof scripts based on verifier feedback.

## Explainer

Translates the final verified proof into readable mathematical explanation, diagrams, and presentation notes.

This project does not implement multiple autonomous agents; these roles describe the MA-LoT-inspired workflow used by the human and AI assistant.
